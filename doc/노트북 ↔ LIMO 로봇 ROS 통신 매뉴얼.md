# 교육 전 노트북 ↔ LIMO 로봇 ROS 통신 매뉴얼

이 문서는 교육 전에 **노트북과 LIMO 로봇이 ROS로 정상 통신되는지 확인하는 방법**을 안내합니다.

---

## 0. 기본 비밀번호 안내

### 노트북 비밀번호

노트북 비밀번호는 **공백 1칸**입니다.

입력 방법은 다음과 같습니다.

1. 비밀번호 입력 화면에서 **스페이스바**를 한 번 누릅니다.
2. **Enter** 키를 누릅니다.

### LIMO 로봇 비밀번호

LIMO 로봇 비밀번호는 다음과 같습니다.

```bash id="twcfgw"
wego
```

입력 후 **Enter** 키를 누릅니다.

---

## 1. 노트북 Wi-Fi 연결하기

노트북의 내장 Wi-Fi 드라이버가 정상적으로 잡히지 않으므로, 제공된 **USB 무선 랜카드**를 노트북에 장착합니다.

USB 무선 랜카드를 장착한 뒤, 강의실에서 제공하는 Wi-Fi에 연결합니다.

---

## 2. LIMO 로봇 부팅 및 Wi-Fi 연결하기

LIMO 로봇을 부팅합니다.

부팅 후 로그인 화면이 나오면, 제공해드린 키보드를 이용하여 비밀번호를 입력합니다.

```bash id="xfe3mq"
wego
```

입력 후 **Enter** 키를 누릅니다.

로그인 후 Wi-Fi 연결을 진행합니다.

LIMO 로봇 화면에서 터치로 Wi-Fi 비밀번호를 입력하는 것이 불편할 수 있으므로, **제공해드린 키보드를 사용하여 Wi-Fi 비밀번호를 입력**합니다.

Wi-Fi 연결 순서는 다음과 같습니다.

1. LIMO 로봇 화면에서 강의실 Wi-Fi를 선택합니다.
2. Wi-Fi 비밀번호 입력 창이 나오면 제공된 키보드로 비밀번호를 입력합니다.
3. 입력 후 **Enter** 키를 누르거나 연결 버튼을 선택합니다.
4. Wi-Fi가 정상적으로 연결되었는지 확인합니다.

---

## 3. 노트북의 `.bashrc` 파일 설정하기

노트북에서 터미널을 열고 `.bashrc` 파일을 수정합니다.

```bash id="409wqh"
gedit ~/.bashrc
```

파일의 가장 아래쪽에 다음 내용을 추가합니다.

```bash id="uv8a53"
export ROS_MASTER_URI="http://로봇IP:11311"
export ROS_IP=노트북고정IP
```

예시는 다음과 같습니다.

```bash id="78ql6o"
export ROS_MASTER_URI="http://192.168.0.10:11311"
export ROS_IP=192.168.0.20
```

여기서 `로봇IP`는 LIMO 로봇의 고정 IP 주소이고, `노트북고정IP`는 노트북의 고정 IP 주소입니다.

내용을 추가한 뒤 파일을 저장합니다.

저장 후 열려 있는 터미널 창을 모두 닫고 다시 실행합니다.

또는 아래 명령어를 실행하여 설정을 바로 적용할 수 있습니다.

```bash id="ihj4v8"
source ~/.bashrc
```

---

## 4. LIMO 로봇의 `.bashrc` 파일 설정하기

LIMO 로봇에서도 터미널을 열고 `.bashrc` 파일을 수정합니다.

```bash id="pxn5kp"
gedit ~/.bashrc
```

파일의 가장 아래쪽에 다음 내용을 추가합니다.

```bash id="7hc8es"
export ROS_MASTER_URI="http://로봇IP:11311"
export ROS_IP=로봇고정IP
```

예시는 다음과 같습니다.

```bash id="3yk6ab"
export ROS_MASTER_URI="http://192.168.0.10:11311"
export ROS_IP=192.168.0.10
```

여기서 `로봇IP`와 `로봇고정IP`는 모두 LIMO 로봇의 고정 IP 주소입니다.

내용을 추가한 뒤 파일을 저장합니다.

저장 후 열려 있는 터미널 창을 모두 닫고 다시 실행합니다.

또는 아래 명령어를 실행하여 설정을 바로 적용할 수 있습니다.

```bash id="374fj2"
source ~/.bashrc
```

---

## 5. LIMO 로봇에서 ROS 실행하기

LIMO 로봇의 터미널 창에서 아래 명령어를 실행합니다.

```bash id="wz6ip1"
roslaunch limo_bringup limo_start.launch
```

이 명령어는 LIMO 로봇을 ROS에서 사용할 수 있도록 실행하는 명령어입니다.

이 명령어는 LIMO 로봇 쪽에서 먼저 실행되어 있어야 합니다.

---

## 6. 노트북에서 키보드 조종 프로그램 실행하기

노트북의 터미널 창에서 아래 명령어를 실행합니다.

```bash id="czviag"
rosrun teleop_twist_keyboard teleop_twist_keyboard.py
```

명령어를 실행하면 노트북 화면에 로봇을 조종할 수 있는 키 안내가 표시됩니다.

화면에 표시된 조작 키를 눌러 LIMO 로봇이 움직이는지 확인합니다.

---

## 7. 통신 테스트 완료 확인

노트북 키보드로 LIMO 로봇이 정상적으로 움직이면, 노트북과 LIMO 로봇 간 ROS 통신 테스트가 완료된 것입니다.

---

## 참고 사항

ROS 통신이 되지 않는 경우 아래 내용을 먼저 확인합니다.

1. 노트북과 LIMO 로봇이 같은 Wi-Fi에 연결되어 있는지 확인합니다.
2. 노트북에 제공된 USB 무선 랜카드가 장착되어 있는지 확인합니다.
3. LIMO 로봇이 강의실 Wi-Fi에 정상적으로 연결되어 있는지 확인합니다.
4. 노트북 IP와 로봇 IP가 올바르게 입력되었는지 확인합니다.
5. `.bashrc` 파일 수정 후 터미널을 새로 열었는지 확인합니다.
6. LIMO 로봇에서 `roslaunch limo_bringup limo_start.launch` 명령어가 먼저 실행되어 있는지 확인합니다.
7. 노트북의 `ROS_MASTER_URI`에 입력한 IP가 반드시 LIMO 로봇 IP인지 확인합니다.
